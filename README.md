# E-Commerce Sales Analysis
## Project Overview
This project presents an end-to-end analysis of an E-Commerce Sales dataset using Microsoft Excel. The analysis focuses on data cleaning, exploratory data analysis (EDA), and chart visualizations to uncover meaningful business insights that can support data-driven decision-making.
## Objectives 
- Clean and prepare raw sales data.
- Perform exploratory data analysis (EDA).
- Identify sales trends and customer purchasing patterns.
- Analyze product performance and revenue.
- Visualize key insights using Excel charts.
## Tools Used 
- Microsoft Excel
- Power Query
- Pivot Tables
- Pivot Charts
## Data Cleaning
The dataset was cleaned using Power Query by: 

- Removing duplicate records.
- Handling missing values.
- Correcting data types.
- Preparing the data for analysis.
